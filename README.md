## Compile and run

- The code is ready to run SPEC CPU Benchmarks, install them and change the path to the benchmarks in the scheduler code before compile.

- Use libperf library for C to compile the scheduler [IBS.c](https://github.com/mlurbe97/Intelligent-Bandwidth-Shifting/blob/master/IBS.c).

- To run the scheduler just launch this script as super user -> [launch_ibs.sh](https://github.com/mlurbe97/Intelligent-Bandwidth-Shifting/blob/master/launch_ibs.sh).
